API DOCUMENTATION

AzLyrics API that returns a promise in JSON format. Another feature of this API is that it prevents the anti-bot detection so we can avoid human verifications such as captchas.

HOW TO GET API KEYS?

Simply click one of the links below, create a new account and get your api key.

ScrapingBee
ZenRows
ScrapeOps

API BASE URL

https://azlyricsapi.hackmesenpai1.repl.co

API ROUTES

Method: GET

/Docs -> documentation page for this api. (no params.)

Method: GET

/GetLyrics -> returns a json response that contains the lyrics.

/GetLyrics Params

api -> your api key
query -> song/music title
url -> song/music link(only azlyrics pages)

/GetLyrics?api="your-key"&query="song-name"
or
/GetLyrics?api="your-key"&url="song/music-link"
